LogEvents
=========

``(implementation of LogEvent entity)``

Provides following method for `API <index.html>`_ calls:

    * `Get logEvent`_
    * `Get logEvents`_
    * `Save logEvent`_
    * `Save logEvents`_
    * `Update logEvent`_
    * `Delete logEvent`_
    * `Get logEvent by entity id`_

.. _`Get logEvent`:

Get logEvent
------------

URL:
~~~~
    */logevents/{id}*

Method:
~~~~~~~
    *GET*

Parameters request:
~~~~~~~~~~~~~~~~~~~
    *null*

Parameters response:
~~~~~~~~~~~~~~~~~~~~
    *Object*

    *With properties:*
        #. id(NUMBER)
        #. timestamp(NUMBER)
        #. entity_class_name(STRING)
        #. entity_id(NUMBER)
        #. field_name(STRING)
        #. previous_value(STRING)
        #. new_value(STRING)
        #. user(OBJECT)
           type of `User <http://docs.ivis.se/en/latest/api/user.html>`_
        #. action(STRING)
            Only can be "CREATE", "MODIFY", "DELETE"

Example of response:
~~~~~~~~~~~~~~~~~~~~

.. code-block:: json

    {
      "id" : 0,
      "timestamp" : 1477474353117,
      "entity_class_name" : "",
      "entity_id" : 0,
      "action" : "CREATE",
      "field_name" : "",
      "previous_value" : "",
      "new_value" : "",
      "user" : {
        "id" : null,
        "name" : null,
        "person" : null,
        "roles" : [ ],
        "saml2_id" : null
      }
    }

Parameters request:

~~~~~~~~~~~~~~~~~~~~

~~~~~~~~~~~~~~~~~~~~~~~~~~
